# 🧮 Console-Based Scientific Calculator

[![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-blue?style=for-the-badge)](LICENSE)
[![GitHub stars](https://img.shields.io/github/stars/HamidFaryad/Console-Base-Scientific-Calculator?style=for-the-badge)](https://github.com/HamidFaryad/Console-Base-Scientific-Calculator)

**A clean, menu-driven scientific calculator that runs directly in your terminal.**  
Perfect for quick math on the go — no GUI, no bloat, just pure Python power!  

![Demo](https://github.com/user-attachments/assets/de899a6f-d9c7-41ce-a423-00bb51a29f4a)

---

## ✨ Features

- **Basic Arithmetic**: Addition, Subtraction, Multiplication, Division
- **Powers & Roots**: Square, Cube, Square Root, Cube Root
- **Trigonometry**: sin, cos, tan + their inverses (asin, acos, atan)
- **Logarithms**: Natural log (ln)
- **Interactive Menu**: Easy number-based selection
- **Error Handling**: Graceful invalid input detection
- **Lightweight & Fast**: Runs instantly in any terminal

Built with ❤️ using only Python's `math` and `cmath` modules.

---

## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/HamidFaryad/Console-Base-Scientific-Calculator.git
cd Console-Base-Scientific-Calculator
```

### 2. Run the Calculator
```bash
python main.py
```

That's it! The beautiful menu will appear instantly.

## 📋 Supported Operations

| Option | Operation              | Example Input          | Sample Output      |
|--------|------------------------|------------------------|--------------------|
| 1      | Addition               | 5 + 3                  | 8                  |
| 2      | Subtraction            | 10 - 4                 | 6                  |
| 3      | Multiplication         | 6 × 7                  | 42                 |
| 4      | Division               | 15 ÷ 3                 | 5.0                |
| 5      | Square                 | 8²                     | 64                 |
| 6      | Cube                   | 3³                     | 27                 |
| 7      | Square Root            | √16                    | 4.0                |
| 8      | Cube Root              | ∛27                    | 3.0                |
| 9      | Sin (radians)          | sin(0)                 | 0.0                |
| 10     | Cos (radians)          | cos(π)                 | -1.0               |
| 11     | Tan (radians)          | tan(π/4)               | 1.0                |
| 12     | Arcsin                 | asin(0.5)              | 0.5236...          |
| 13     | Arccos                 | acos(0)                | 1.5708...          |
| 14     | Arctan                 | atan(1)                | 0.7854...          |
| 15     | Natural Log (ln)       | ln(2.718)              | 1.0                |
| 16     | Exit                   | —                      | Program Closed     |

**Note**: All inputs are currently integers. Trigonometric functions use **radians** (not degrees). Future updates will add degree mode & floating-point support!
